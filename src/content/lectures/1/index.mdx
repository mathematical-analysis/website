## Lecture

### Real Numbers Introduction

Real numbers are integral to mathematics. Standard notations include:

- $\emptyset$ is the empty set;
- $\mathbb{N} = \{1, 2, \dots\}$, the set of natural numbers;
- $\mathbb{Z} = \{\dots, -1, 0, 1, \dots\}$, the set of integers;
- $\mathbb{Q} = \left\{ \frac{p}{q} \mid p, q \in \mathbb{Z}, q \neq 0 \right\}$, the set of rational numbers;
- $\mathbb{R}$, the set of real numbers.

### Bounds and Extrema

**Definition 1.1:** Let $S$ be a subset of $\mathbb{R}$, denoted as $S \subseteq \mathbb{R}$. We define:

- a **lower bound** for $S$ if $g \leq x, \forall x \in S$;
- an **upper bound** for $S$ if $g \geq x, \forall x \in S$;
- the set of lower bounds of $S$, $L(S) = \{g \in \mathbb{R} \mid g \leq x, \forall x \in S\}$;
- the set of upper bounds of $S$, $U(S) = \{g \in \mathbb{R} \mid g \geq x, \forall x \in S\}$.

**Definition 1.2:**

- The **minimum** of $S$, if it exists, is the greatest element of $L(S)$ that is in $S$;
- The **maximum** of $S$, if it exists, is the least element of $U(S)$ that is in $S$.

### Supremum and Infimum

**Definition 1.3:** $g \in \mathbb{R}$ is the **supremum** of $S \subseteq \mathbb{R}$, $g := \sup(S)$, if:

1. $g \geq x, \forall x \in S$;
2. for any upper bound $d$ of $S$, $g \leq d$.

**Definition 1.4:** $g \in \mathbb{R}$ is the **infimum** of $S \subseteq \mathbb{R}$, $g := \inf(S)$, if:

1. $g \leq x, \forall x \in S$;
2. for any lower bound $d$ of $S$, $g \geq d$.

### Completeness Axiom

**Definition 1.5 (Completeness Axiom):** Every non-empty set $S \subseteq \mathbb{R}$ that is bounded above has a supremum in $\mathbb{R}$. Similarly, every set that is bounded below has an infimum.

### Neighborhoods and Interior/Closure

**Definition 1.10:** A set $U \subseteq \mathbb{R}$ is a **neighborhood** of $g \in \mathbb{R}$ if there exists $\epsilon > 0$ such that $(g - \epsilon, g + \epsilon) \subseteq U$.

**Definition 1.11:**

- The **interior** of $S$, $\operatorname{int}(S) = \{g \in \mathbb{R} \mid \exists U \text{ neighborhood of } g \text{ such that } U \subseteq S\}$;
- The **closure** of $S$, $\operatorname{cl}(S) = \{g \in \mathbb{R} \mid \forall U \text{ neighborhood of } g, U \cap S \neq \emptyset\}$.

## TLDR;

Real numbers are fundamental in mathematics, categorized and defined as follows:

- **Standard Sets**:

  - $\emptyset$: the empty set.
  - $\mathbb{N}$: the set of natural numbers, $\{1, 2, \dots\}$.
  - $\mathbb{Z}$: the set of integers, $\{\dots, -1, 0, 1, \dots\}$.
  - $\mathbb{Q}$: the set of rational numbers, $\left\{ \frac{p}{q} \mid p, q \in \mathbb{Z}, q \neq 0 \right\}$.
  - $\mathbb{R}$: the set of real numbers, encompassing all continuous values.

- **Bounds and Extrema**:

  - **Lower Bound**: A value less than or equal to every element in a subset $S$ of $\mathbb{R}$.
  - **Upper Bound**: A value greater than or equal to every element in $S$.
  - **Minimum** and **Maximum**: The greatest lower and least upper bounds of $S$ that belong to $S$, if they exist.

- **Supremum (Sup) and Infimum (Inf)**:

  - **Supremum**: The least upper bound of a set $S$.
  - **Infimum**: The greatest lower bound of a set $S$.

- **Completeness Axiom**:

  - Every non-empty set in $\mathbb{R}$ that is bounded above has a supremum.
  - Every set in $\mathbb{R}$ that is bounded below has an infimum.

- **Neighborhoods and Interior/Closure**:
  - **Neighborhood**: A set around a point in $\mathbb{R}$ within a certain radius.
  - **Interior**: All points in a set that have a neighborhood completely within the set.
  - **Closure**: All points where every neighborhood intersects with the set.
